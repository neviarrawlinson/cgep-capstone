# Compliant S3 Primitive

This module creates a compliant AWS S3 storage primitive for CGE-P Lab 2.3. It enforces NIST 800-53 aligned controls for SC-28, AU-3, AU-6, CM-6, and AC-3 by enabling AES-256 server-side encryption, versioning, access logging, full public access blocking, and required compliance tags. Evidence is generated through Terraform plan and state JSON output instead of screenshots.
